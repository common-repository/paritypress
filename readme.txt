## How it Works?

The ParityPress plugin detects automatically the country of the website visitor. Then the plugin dynamically showcases the pricing information that you've preconfigured for that specific country.

This means that when a user accesses your website, the plugin automatically recognizes where they are connecting from in real-time.

== Installation ==

The simplest way to install ParityPress is by searching for it in your website's Dashboard.

**How to Install ParityPress on WordPress**
1. Visit your WordPress Dashboard.
2. Navigate to the "Add New Plugin" section.
3. Search for "ParityPress."
4. Click on "Install Now" to install the plugin.
5. After installation is complete, click "Activate Plugin."
